# h5c-new

- 使用 vuex 1.x
- 使用 eslint 对代码格式进行监测


> a h5 online editor

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
